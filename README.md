## Running locally

Clone the project

```bash
  git clone https://github.com/felipefreitasa/nft-app-animation.git
```

Enter the project directory

```bash
  cd nft-app-animation
```

Install the dependencies

```bash
  yarn
```

Start the server

```bash
  npx expo start
```
